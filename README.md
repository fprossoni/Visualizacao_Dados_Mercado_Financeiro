# Visualização de Dados do Mercado Financeiro
Esse Notebook Jupyter foi desenvolvido para a visualização e análise de dados de diferentes ativos do mercado financeiro, incluindo o Ibovespa, o S&P 500, o dólar comercial (USD/BRL), o CDI e algumas ações da B3.
A proposta é comparar rentabilidade, volatilidade, volume de negociação e comportamento dos ativos ao longo do tempo, além de observar possíveis relações com o cenário macroeconômico e entre setores.  
O notebook foi construído principalmente com as bibliotecas pandas, para organização e tratamento dos dados, e altair, para a criação dos gráficos interativos. 
Na etapa de preparação dos dados, os arquivos foram formatados e padronizados para permitir a visualização correta nos gráficos. Os dados de ações e índices foram obtidos do Yahoo Finance, enquanto os dados do CDI foram obtidos do Sistema Gerenciador de Séries Temporais do Banco Central do Brasil.
Para o funcionamento correto do notebook, é necessário fazer o upload de todos os arquivos .csv presentes na pasta do projeto (no caso de utilização pelo Google Colab).
